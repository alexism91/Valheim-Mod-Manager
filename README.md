# Valheim Mod Manager

A native GTK4 Linux application to manage mods for the game Valheim.
Mods are fetched from [Thunderstore.io](https://thunderstore.io/).

## Features

- Browse available mods on Thunderstore
- One-click install and uninstall
- Manage updates for installed mods
- Immersive "Valheim" theme
- Local cache management (icons and downloads)

### Dependencies (manual install)

- Python 3.10+
- GTK4 & libadwaita
- PyGObject

### Run from source

```bash
python3 main.py
```
