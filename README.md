# Valheim Mod Manager

Application GTK4 native pour Linux permettant de gérer les mods pour le jeu Valheim.
Les mods sont récupérés depuis [Thunderstore.io](https://thunderstore.io/).

## Fonctionnalités
- Parcourir les mods disponibles sur Thunderstore.
- Installation et désinstallation en un clic.
- Gestion des mises à jour des mods installés.
- Thème immersif "Valheim Professionnel".
- Gestion du cache locale (icônes et téléchargements).

## Installation

### Dépendances
- Python 3.10+
- GTK4 & libadwaita
- PyGObject

### Lancement
```bash
python3 main.py
```
